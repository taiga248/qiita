# ゼロからのマークダウン記法
## まずマークダウン記法ってなに？
自動的に文章の**見栄えを良く**してくれる記法

## できること
1.  見出し
2.  語句を**強調**
3.  [リンクを貼る](https://github.com/taiga248/Markdown)
4.  ``` プログラミングコードの記述 ```
5.  リスト化
6.  etc..
